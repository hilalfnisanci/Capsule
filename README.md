# Capsule

A personal memory and knowledge management app — your private digital capsule for storing notes, links, images, and reflections over time.

## Product Description

Capsule lets you capture and revisit moments, ideas, and discoveries. Think of it as a personal archive: a place to drop anything worth remembering and find it again later, organized by tags, time, or full-text search.

## Core MVP Idea

A lightweight web app where a single user can:
- Save entries (text notes, links, images, short reflections)
- Tag and search entries
- Browse entries in a chronological feed or timeline view

## Planned Main Features

- **User authentication** — secure sign-up, login, and session management
- **Capsule entries** — create, read, update, and delete notes, links, and media
- **Tag-based organization** — apply one or more tags to any entry
- **Full-text search** — find entries by content or tags
- **Timeline / feed view** — browse entries chronologically
- **Responsive web UI** — works on desktop and mobile
